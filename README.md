# AWS_Cloud_Project [ SEEDSHOP ]
 AWS를 활용한 클라우드 프로젝트
## 1. 기술 
- 언어
  - `HTML/CSS`
  - `JavaScript`
  - `Java(JDK 13.0.2)`
  - `JSP`
 - AWS
   - `EC2` : WAR배포
   - `RDS` : Oracle Enterprise Edition
   - `SES`
